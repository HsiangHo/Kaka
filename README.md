去掉了瑞星无用的杀毒和防火墙功能,只保留小狮子。此版本由Windows移植至macOS,可当桌面宠物饲养,并增加实用功能。

## Features

1. 一键显示/隐藏桌面图标。(增加对多屏幕支持)
2. 一键返回桌面。
3. 自动(3秒)隐藏鼠标光标。
4. 防止系统自动休眠（支持合盖）。
5. 自动(10秒)隐藏桌面图标。
6. 自动(根据屏幕亮度)切换黑暗模式。
7. 一键休眠，关屏幕，屏幕保护。
8. 自定义App的显示模式。
9. 禁用鼠标。  
  
10.捷径中心。
